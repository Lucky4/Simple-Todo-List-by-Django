# Simple-Todo-List-by-Django

#Overview

  A simple web application called Todo List.

#Requirements
* Windows7
* Python2.7
* Django1.8
* MySQL
* mysql-python1.2.5
* djangorestframework3.6.2
* Bootstrap
* virtualenv
  

#Done
1. 增加一个待办事项
2. 删除一个待办事项
3. 标记一个待办事项为已完成
4. 编辑一个待办事项的具体内容
5. 列出所有的待办事项
6. 列表界面支持翻页
7. 待办事项可以设置优先级
8. 待办事项可以设置expire date
9. 支持按照不同的方式排序，如优先级，expire date
